## SATA ATTENDANCE WEB APPLICATION

### Features: 
- Tracks, Strand, Levels and Section Management
- Check Attendance
- School Updates
- Attendance Result
- Class Records
- Activity Logs
- Send SMS

### Technologies
- React JS
- React Native
- PHP Laravel
- MySQL
